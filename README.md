# Projeto To-Do

Projeto desenvolvido durante estudos de Node.js, aplicando conceitos de MVC, Express, Sequelize e MySQL para criação de uma aplicação completa de gerenciamento de tarefas.

 ## Tecnologias utilizadas

- Node.js
- Express
- Express-Handlebars
- Sequelize
- MySQL
- Nodemon
- Arquitetura MVC
